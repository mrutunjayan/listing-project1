# 🏡 Listing Project

A full-stack web application where users can create, view, edit, and delete listings (like Airbnb-style properties). This project demonstrates CRUD operations, authentication, and image handling.

---

## 🚀 Features

- 📝 Create new listings
- 📋 View all listings
- ✏️ Edit existing listings
- ❌ Delete listings
- 🔐 User authentication (Login / Register)
- 🖼️ Image upload support
- 💬 Flash messages for user actions

---

## 🛠️ Tech Stack

- **Frontend:** EJS, CSS, Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** Passport.js
- **Image Storage:** ImageKit
